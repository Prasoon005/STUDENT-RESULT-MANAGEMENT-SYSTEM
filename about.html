Student Marks Management System

Effortlessly manage and track student marks

Introduction

Welcome to the Student Marks Management System! This web-based application has been designed to simplify the management of student academic records. It allows educators to input, manage, and view student marks in various subjects, such as Maths, English, and Science. With this system, teachers and administrators can efficiently calculate total marks, monitor student performance, and maintain organized records.

Key Features:

  • Input student names and marks for different subjects.
  • Automatically calculate the total marks for each student.
  • View results in a simple, organized table.
  • Easy-to-use interface with clear and intuitive design.

This system ensures that tracking student progress is fast, simple, and effective, allowing you to focus more on educational growth rather than administrative tasks.

System Features

Data Entry

Easily input student details and marks in multiple subjects.

Automatic Calculation

Automatically calculate total marks for each student based on their subject scores.

Result Management

View student results instantly and manage them efficiently in a table format.



Technologies Used

This system was built using the following technologies:

HTML

HTML (HyperText Markup Language) provides the structure of the webpage, including elements like headings, paragraphs, forms, and tables. It's the foundation that supports the content of the Student Marks Management System.

CSS

CSS (Cascading Style Sheets) is used for styling the webpage. It controls the layout, colors, and positioning of elements. Transitions and animations in CSS were used to enhance the user interface and make the page more interactive.

JavaScript

JavaScript adds interactivity to the system. It handles the logic for submitting student marks, calculating totals, and dynamically updating the results table. This makes the system functional and user-friendly.

Figma

Figma was used to design the layout and user interface of the system. It helped in creating a modern, clean design and ensures that the final webpage is responsive and visually appealing.
